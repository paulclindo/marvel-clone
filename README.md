## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes.

### Pre-requisites

None

### Installing

Install the dependencies:

```
npm i
```

Run project:

```
npm run start
```

### Technologies

This was built using HTML, CSS and vanilla JS.
